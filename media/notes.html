Elliot Gong: Portfolio Design Process

Right now, my main goal in creating my portfolio page is have
somewhere to displaymy work while also applying my HTML and CSS skills.
My web development experience is still a work in progress, so my desired
audience is the average web user as they won't be expecting too much from
a page like this. Using new techniques and tools for this assignment is also
useful as I can get more exposure to the more advancedaspects of web design.
Even though this is just a prototype, I want to make this as close to the final
version as possible.
Now, I've made some edits and added my custom content to the page, so it's more polished. I'll move it out of the 235 folder later, and perhaps into an external service.

As you can see, my design sketches for my web page is fairly basic, but that's
the intention. I want to keep things simple, and just get the point across as this
is my own personal page. Of course, I want to make my website as accessible and
responsive as it can be, and be easy to figure out and navigate.
